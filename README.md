## Development workflow demo
A simple web application developed as part of the SoftUni DevOps for Developers course.

## Project Overview

This project demonstrates a complete development workflow using:

- Git & GitHub
- GitHub Projects (Kanban Board)
- Feature Branching
- Pull Requests
- Code Review
- QA Verification
- Staging & Production Deployments

## Features

- Change Background
- Add Genre Field
- Add Field Icons

## Workflow

- Create issue in GitHub Projects
- Move issue to In Progress
- Create feature branch
- Implement feature
- Commit and push changes
- Merge feature branch into qa
- QA verification
- Create Pull Request to staging
- Code review
- Merge staging into main
- Deploy to Production
